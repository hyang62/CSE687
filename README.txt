# CSE687

Backend-Calculator

How to use the calculator program?

1. Unzip the "cse687" zipped file.
2. Open the Calculator.java file under the folder "\priceCalculator\src\Calculator", and execute the file.
3. At line 497 and line 501, you can modify the desire cities, and at line 510, you can modify the departure date.
4. The log files will be generated under the folder "\priceCalculator" with the name in the format as "departer to arrival", "LAXtoDEN" for instance.
